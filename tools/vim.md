# vim高亮多个搜索结果

## 步骤如下：
1. 首先键入搜索模式，即在正常模式下键入“/”
2. 在搜索模式下键入“\v”  
3. 把你要搜索的多个关键字用“|”相连，比如"Excluding|System"，回车  

# 在同一个窗口打开多个文件
> vi file1 file2
vim命令模式下
:n 切换到下一个文件
:N 切换到上一个文件

# 在不同窗口打开多个文件

1. 先使用vi打开一个文件，例如vi file1
2. 分割窗口打开另外的文件
> :sp file2 //水平分割打开第二个文件，sp=split
:vsp file2 //垂直分割打开第二个文件，vsp=vsplit
:vs 也可

3. 窗口切换：可以使用 ctrl +两次按 w，这样就可以在各个文件中切换。
4. 设置窗口宽度
> :vertical resize 110
